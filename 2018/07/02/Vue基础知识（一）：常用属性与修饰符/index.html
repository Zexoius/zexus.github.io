Vue常用选项
  1. data

    每个 Vue 实例都会代理 data 对象里所有的属性。

    当一个组件被定义， data 必须声明为返回一个初始数据对象的函数，因为组件并不可能只用于创建一个实例，而可能是多个。

    因此，通过提供 data 函数，每次创建一个实例后，我们能够调用此函数以返回初始数据的的一个全新副本数据对象。
    简而言之，每个组件实例都有自己的作用域，每个实例相互独立，不会相互影响。

常用实例属性
  1. el表示当前挂载点。
  2. props表示当前组件接收到的 props 对象。Vue 实例代理了对其 props 对象属性的访问。
  3. data获取当前实例上挂载的数据。

  4. options当前实例上自定的属性。用于当前 Vue 实例的初始化选项。需要在选项中包含自定义属性时会有用处：

  5. children当前实例的直接子组件。需要注意 $children 并不保证顺序，也不是响应式的。如果你发现自己正在尝试使用 $children 来进行数据绑定，考虑使用一个数组配合 v-for 来生成子组件，并且使用 Array 作为真正的来源。

常用实例方法/数据
  1. watch 观察 Vue 实例变化的一个表达式或计算属性函数。回调函数得到的参数为新值和旧值。表达式只接受监督的键路径。对于更复杂的表达式，用一个函数取代。
